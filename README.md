# Yotter
The Botframework for perfectionists with deadlines.

## Start project
```
python3 -m Yotter startproject Test
```

---

## Modules
| Module | Purpose |
|---|:--|
| config | setup bot |
| handlers  | creating and register handlers |
| middlewares  | creating and register middlewares  |
| manage.py | manager script |

---

## Run Bot
```
python3 manage.py runbot
```
